# План развития Call Calendar (OpenCode)

Ниже перечислены задачи, которые будут выполняться с участием агента OpenCode через issues и pull requests.

## Баги

1. **Пересечение временных слотов при бронировании**  
   Система иногда позволяет забронировать пересекающиеся слоты. Нужно добавить проверку пересечений и тесты.

2. **Некорректный сдвиг часовых поясов при отображении**  
   Если гость и владелец в разных часовых поясах, время может отображаться неверно.

## Фичи

3. **Валидация email при создании гостя**  
   Добавить проверку формата email на бэкенде и понятное сообщение об ошибке.

4. **Повторяющиеся события (recurring events)**  
   Возможность создавать ежедневные/еженедельные бронирования по правилу.

5. **Уведомления по email о бронировании**  
   Отправка письма гостю и владельцу после создания брони.

6. **Регулярные ночные проверки Lighthouse**  
   Настроить scheduled GitHub Actions workflow для запуска Lighthouse-аудита страниц календаря и сохранения отчёта.

## Инфраструктура

7. **Интеграция OpenCode GitHub App**  
   Настройка агента для автоматического анализа issues, создания PR и участия в ревью.
